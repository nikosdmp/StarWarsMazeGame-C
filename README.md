# Star Wars Maze Game (C)

A console-based Star Wars game where you navigate **Leia** through obstacles, stormtroopers, and **Darth Vader**.

## Features
- Move Leia with keyboard commands (U/D/L/R)
- Avoid stormtroopers (@) and obstacles (X)
- Defeat levels by reaching R2D2 (R)
- Levels become progressively harder

## Tech Stack
- C (Console-based)

## How to Run
Compile and run using GCC:

```bash
gcc main.c -o StarWarsGame
./StarWarsGame
```
